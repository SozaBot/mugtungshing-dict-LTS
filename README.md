## 贡献

欢迎贡献！

1. 点击右上角的 Fork
2. 创建一个分支 (`git checkout -b feature/user-update-dict`)
3. 提交更改 (`git commit -m 'feat(dict): add SDVX'`)
4. 推送更改 (`git push origin feature/user-update-dict`)
5. 提交 Pull Request

> 需要向词典添加内容，请参见 [`dict.json`](./src/tungshing/dict.json)
